  2.  Qu'est ce que OmegaT ?

OmegaT est un outil d'aide  la traduction. OmegaT est distribu gratuitement, il n'est donc pas ncessaire de payer de licence d'utilisation mme pour un usage professionnel. OmegaT est une application libre, vous avez donc le droit de la modifier et/ou de la redistribuer tant que vous respectez les termes de la licence utilisateur.

Les principales fonctions de OmegaT sont:
  - capacit de fonctionner sur n'importe quel systme d'exploitation qui supporte Java
  - utilisation de n'importe quel type de fichier TMX conforme comme mmoire de traduction de rfrence
  - systme de segmentation de texte flexible (bas sur le standard SRX)
  - recherche de termes dans le projet et dans les mmoires de rfrence
  - recherche de terms dans n'importe quel rpertoire incluant des fichiers lisibles par OmegaT
  - correspondances automatiques
  - gestion intelligente des projets incluant des structures complexes de rpertoires
  - glossaires (vrifications terminologiques)
   - documentation facile  lire et tutoriel simple
   - localisation dans de nombreuses langues

OmegaT peut vous aider  traduire des fichiers aux formats suivants : OpenDocument, Microsoft Office ( l'aide de OpenOffice.org utilis comme filtre de conversion, ou aprs conversion en HTML), OpenOffice.org ou StarOffice, (X)HTML, fichiers de localisation Java, fichiers textes, DocBook et PO.

OmegaT est capable d'analyser les structures de rpertoires les plus complexes pour y retrouver tous les fichiers lisibles. OmegaT recrera la mme structure de rpertoires pour vos documents traduits et y inclura une copie de tous les fichiers qu'il n'a pas pu lire.

Pour commencer  utiliser OmegaT tout de suite, lancez OmegaT et lisez le tutoriel qui s'affiche dans la fentre principale : �OmegaT : Pour commencer tout de suite�.
