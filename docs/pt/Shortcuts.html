Keyboard Shortcuts

The shortcuts given here are for those operating systems with a Control (Ctrl) key (UNIX, Windows, etc.). For Mac OS X use the Apple key (cmd) in place of Ctrl. On other systems the corresponding key will have to be located by the user. In some instances OmegaT shortcuts will not work and the mouse will have to be used.

Common shortcuts are available to select strings in all of OmegaT's windows (e.g. to cut and paste within an active segment) from the default ones on a particular operating system. See the operating systems documentation for more details.

File Menu
<Ctr+O> Open Project...
<Ctr+L> Project Files... (open Project Files window)
<Ctr+M> Match/Glossary... (open/close Match and Glossary Viewer window)
<Ctr+S> Save Project
<Ctr+Q> Quit
Edit Menu
<Ctr+Z> Undo Last Action
<Ctr+Y> Redo Last Action
<Ctr+R> Overwrite Translation (replace text in active segment with selected fuzzy match)
<Ctr+I> Insert Translation (insert selected fuzzy match into active segment at cursor position)
<Ctr+F> Search Project... (open Search window)
<Ctr+U> Next Untranslated Segment
<Ctr+N>,
<Enter>,
<Tab>*		 Next segment. *(to use <Tab> it must be configured for use. See Translation Process for details)
<Ctr+P> Previous segment
<Ctr+1, 2, etc.> Select fuzzy match 1, fuzzy match 2, etc.
Help Menu
F1 Open Users Manual
