Java Runtime Environment

Program OmegaT vyžaduje mať nainštalované prostredie Java Runtime Environment. (tiež známe ako JRE). V podstate, ak môžete na vašom systéme spúšťať programy Java tak budete môcť spustiť aj OmegaT. Prostredia Java Runtime Environment sú dostupné pre nasledujúce platformy: Windows 95/98/2000/ME/XP, UNIX, Linux, a Solaris (x86 aj Sun SPARC). Mac OS X ho má predinštalované.

Počítač na ktorom chceme spustiť OmegaT musí mať nainštalované prostredie Java Runtime Environment(JRE) —BEZ NEHO NEBUDE FUNGOVAŤ!

Java a OmegaT možno inštalovať v ľubovoľnom poradí. Na UNIXOV-ských platformách konfiguirácia nie je nutná, ale umožní spustenie aplikácie používateľsky príjemnejším spôsobom tým ktorí to požadujú.

Ak si nie ste istí, či je prostredie JRE nainštalované, nainštalujte najprv OmegaT a potom otestujte či funguje (pozri Inštalácia OmegaT). Ak sa nič nestane, alebo dostanete hlásenia o chybe, potom takme rurčite nie je prostredie Java Runtime Environment nainštalované. V tom prípade, prosím zaobstarajte si ho a najprv ho nainštalujte.

Prosím všimnite si, že prostredie JRE je dostupné vo forme samostatného produktu (J2RE) alebo ako súčasť Java Software Developer's Kit(J2SDK).

Toto vydanie OmegaT vyžaduje prostredie JRE verzie 1.4 alebo novšie.

Pokiaľ si je tím projektu OmegaT vedomý, OmegaT NEbeží na nasledujúcich platformách:

  • Windows NT; NT implementácia Javy obsahuje chyby
  • Mac OS 9; jediné prostredie Java Runtime Environment preň dostupné (známe ako MRJ) je príliš staré a neposkytuje požadovanú funkčnosť.

Inštalácia prostredia Java

Dôležité poznámky:

Sun pravidelne mení názvy svojich produktov—možno kvôli tomu aby zmiatli svojich zákazníkov. Nemali by ste sa o to veľmi starať, potrebujete iba posledné prostredie Java Runtime Environment. Verszia 1.4 alebo vyššia, či už to je Java 2 verzia 1.4.2 alebo Java 5 verzia 1.5.0.

Existujú aj iné príbuzné (alebo nie až tak príbuzné) formy prostredia Java:

  • Java pre zahrnuté aplikácie pre webové prehliadače (tj. cez zásuvné moduly - pluginy);
  • Skriptovací jazyk JavaScript pre webové stránky (technicky nemá vzťah k prostrediu Java).

Žiadna z týchto foriem prostredia Java neumožní spustenie OmegaT.

Java na Windows

Skontrolujte ktorá verzia JRE, ak vôbec, je nainštalovaný pomocou (tlačidlo)Štart/klávesa Windows > Nastavenia > Ovládací panel. Ak už je prostredie JRE nainštalované, v ovládacom paneli sa bude nachádzať aplet označený "Java" alebo podobne. Spustite ho a skontrolujte, či má verziu 1.4 alebo vyššiu.

Prosím všimnite si, že v čase písania tohto textu je aktuálna verzia prostredia Java dostupná na Windows označená v rozsahu 1.5 (stabilná verzia). Program OmegaT je testovaný pre prácu s touto verziou.

Verzie 1.4.x programu OmegaT by mali fungovať s akýmkoľvek JRE verzie 1.4 a vyššej.

Používatelia si môžu stiahnuť prostredie Java Runtime Environment od firmy:
Sun na adrese http://java.sun.com/getjava/index.html

Inštalačné pokyny pre JRE možno nájsť na webovej stránke odkiaľ sa dajú stiahnuť alebo pri stiahnutom súbore alebo balíku pri jeho otvorení alebo zozbalení.

Java na systémoch UNIX (vrátane Linux & Solaris)

V príkazovom riadku v konzole alebo v terminálovom programe zadajte java -version. Dostanete číslo nainštalovanej verzie (ake je prostredie Java nainštalované ako spustiteľný súbor vo vašej 'shell path').

Prosím všimnite si, že v čase písania tohto textu je aktuálna verzia prostredia Java dostupná pre Linux označená v rozsahu 1.5 (stabilná verzia). Program OmegaT je testovaný pre prácu s touto verziou.

Verzie 1.4.x programu OmegaT by mali fungovať s akýmkoľvek JRE verzie 1.4 a vyššej.

Používatelia si môžu stiahnuť prostredie Java Runtime Environment od firmy:
Sun na adrese http://java.sun.com/getjava/index.html
Blackdown (Linux) na adrese http://www.blackdown.org/java-linux.html

Inštalačné pokyny pre JRE možno nájsť na webovej stránke odkiaľ sa dajú stiahnuť alebo pri stiahnutom súbore alebo balíku pri jeho otvorení alebo zozbalení.

Java na MacOSX

Používatelia Mac: Na MacOSX, je JRE vopred nainštalované a nastavené. Toto je jediná platforma, ktorá nevyžaduje aby používateľ inštaloval JRE. Ak OmegaT nefunguje, uistite sa, že spustíte z Menu Apple Aktualizáciu softvéru aby ste získali najnovšie JRE dostupné pre vašu verziu OSX. Program OmegaT by mal fungovať na OS X od OS X 10.2 až po najnovšiu aktualizáciu OS X 10.4.3.

Prosím všimnite si, že v čase písania tohto dokumentu je aktuálnou verziou prostredia Java dostupnou na MacOSX Java 1.4.2. Program OmegaT je testovaný pre prácu s touto verziou.

Staršie verzie OmegaT (menovite 1.0) by mali fungovať na MRJ pre Macintosh OS 9. OmegaT verzie 1.4 alebo vyššej na tejto platforme nefunguje.

Pre spustebnie verzie OmegaT 1.0, si stiahnite MRJ pre Mac OS9 z webovej stránky Apple. OmegaT 1.0 je zastaralá verzia a táto dokumentácia sa na ňu už nevzťahuje.

Inštalácia OmegaT

Všetky platformy

Na inštaláciu je k dispozícii generický archív OmegaT. Táto verzia OmegaT funguje na systémoch Linux, Solaris, MacOSX, a Windows (a možno aj na iných). Pozri tiež Systémové požiadavky.

Jednoducho rozbaľtw obsah .zip archívu do ľubovoľného adresára.

Windows

Používatelia Windows si môžu stiahnuť binárny inštalátor pre Windows a jednoducho ho spustiť. V tomto prípade, inštalátor vytvorí odkaz pre spustenie OmegaT z menu Štart > Programy.

MacOSX

Používatelia OSX si môžu stiahnuť balíček pre MacOSX, ktorý automaticky nainštaluje balík pre MacOSX do zadaného adresára. Tento balík obsahuje dokovaciu ikonu a aplikáciu spustiteľnú dvojklikom.

Konfigurácia OmegaT

UNIX

Táto sekcia je pre používateľov systémov UNIX ale aj používatelia príkazového riadku MacOSX z nej môžu mať úžitok.

Ak sa inštalácia na UNIX-ovom systéme robí pomocou generického archívu, a nemáte dobré znalosti o platforme, vykonajte nasledujúce kroky aby ste si zjednodušili spustenie OmegaT:

Uložte nasledujúci obsah Bash shell skriptu ako textový súbor:

#!/bin/bash
cd <OmegaT>
<java>/java -jar OmegaT.jar

kde <OmegaT> je adresár obsahujúci súbor OmegaT.jar a <java> je adresár obsahujúci spustiteľný súbor J2RE (súbor s názvom java). Ak je nastavená premenná cesty Java, nie je potrebné zadávať tento adresár, a java -jar OmegaT.jar stačí na spustenie programu. Uvedenie adresára je potrebné ak premenná cesty k prostrediu Java nie je nastavená, a tiež ak je na systéme nainštalovaných viac verzií prostredia Java a na spustenie OmegaT je potrebná špecifická verzia.

Ak Bash nie je predvoleným shellom zmeňte prvý riadok skriptu tak aby to bolo zohľadnené (napr. pre Korn shell použijte #!/bin/kshnamiesto #!/bin/bash)

Uložte skript ako OmegaT, omegat, alebo po akýmkoľvek názvom chcete. Potom nastavte prístupové práva súboru skriptu na spustiteľné (napr. pomocou$ chmod u+x OmegaT v shelli, alebo pri použití Konquerora, kontextové menu > Vlastnosti > Práva > Spustiteľný). OmegaT potom možno spustiť z príkazového riadku zadaním názvu ktorým sme pomenovali skript. Pre spustenie z pracovnej plochy, vytvorte ikonu a nastavte ju na vykonanie súboru skriptu shellu.

Spustenie OmegaT

Windows

Ak bol program OmegaT nainštalovaný pomocou inštalátora Windows (pozri vyššie), možno ho spustiť kliknutím na odkaz OmegaT z menu Štart > Programy > OmegaT. Tiež možno kliknúť na jeden zo súborov OmegaT.exe, OmegaT.bat, alebo OmegaT.jar v správcovi súborov.

UNIX

Spustite java -jar OmegaT.jar z príkazového riadku. Zachovajte presnú formu vyľkých písmen OmegaT—sú dôležité.

Ak nebolo prostredie Java nastavené na detekciu umiestnenia súborov OmegaT, zmeňte adresáre na adresár obsahujúci súbor OmegaT.jar.

Ak nie premenná cesty Java path nastavená, uveďte najprv celú cestu. Ak nebolo urobené nič z predchádzajúcich vecí, postup pre spustenie aplikácie by mohol vyzerať nejako takto pre inštaláciu z generického archívu:

cd /home/ <pouzivatel>/ <inštalačný adresár OmegaT>
<adresár JRE>/bin/java -jar OmegaT.jar

pre používateľa Joe Blow, inštalačný adresár archívu OmegaT, a adresára JRE

/usr/jre1.5

by to bolo

cd /home/joeblow/OmegaT
/usr/jre1.5/bin/java -jar OmegaT.jar

MacOSX

Ak bol nainštalovaný balík MacOSX, jednoducho kliknite na ikonu OmegaT umiestnenú v inštalačnom adresári. V prípade, že boli stiahnuté a nainštalované iba štandardné súbory Java, jednoducho kliknite na súbor OmegaT.jar v okne Finder.

Program OmegaT možno spustiť aj z Terminálu (podobne ako v príkazovom riadku v Linuxe). Viac informácií nájdete vyššie v sekciách konfigurácie a spustenia pre systémy UNIX.

Zmena nastavenia jazyka používateľského rozhrania

Jazyk použitý pre používateľské rozhranie v OmegaT a používateľský manuál sú nastavené na predvolenú hodnotu, ktorá závisí na vašom používateľskom prostredí a na dostupných lokalizáciách. Ak neexistuje lokalizácia zodpovedajúca vášmu prostrediu bude použitá angličtina. Zoznam dostupných lokalizácií nájdete nižšie - Dostupné jazyky pre OmegaT 1.6.

Akákoľvek platforma, z príkazového riadku

Je možné obísť predvolený jazyk používaný v OmegaT zadaním dvojpísmenkového kódu jazyka, z poskytnutého zoznamu, pre voľbu java -Duser.language (na príkazovom riadku alebo v skripte shellu). Toto môže byť užitočné keď používate OmegaT v prostredí, kde predvolené locale nie je podporované, alebo ak je iné než to ktoré chcete. Napríklad, nasledujúci príkaz spustí OmegaT vo francúzštine bez ohľadu na to či ostatné aplikácie v pracovnom prostredí sú vo francúzštine alebo nie:

java -Duser.language=fr -jar OmegaT.jar

Špecifickejšie nastavenia locale nie sú v súťasnosti pre lokalizáciu OmegaT podporované (tj. pomocou volieb -Duser.country a -Duser.variant pre javu).

Treba však poznamenať fakt, že pre jazyky ako je japončina môže byť potrebné použiť špecifickú internacionalizovanú verziu JRE od spoločnosti Sun pre správnu podporu kódovania znakov. Toto je najpravdepodobnejšie v prípade inštalácie na Windows.

Akákoľvek platforma, pomocou pythonovského skriptu

Ak je na vašom systéme nainštalovaný Python, môžete použiť skript jazyka Python, ktorý poskytuje jednoduché grafické okienko pre výber jazyka. Skript môžete násjsť na OmegaT Yahoo! skupinovej stránke, [Files] sekcia, [Macros and tools]. Skript je OmegaTpy.zip vytvorený Dmitriom Gabinskym.

MacOSX, pomocou nastavení [Predvoľby systému], [Medzinárodné].

Na MacOSX, možno zmeniť locale z Predvolieb systému v Menu Apple zmenou obsahu záložky predvolieb Medzinárodné. V záložke [Jazyk], potiahnite jazyk, ktorý chcete pre spúšťanie OmegaT na vrch zoznamu. Ak jazyk nie je na zozname, kliknite na [Editovať zoznam...] a pridajte jazyk. Spustite OmegaT so zmeneným nastavením a keď je program OmegaT spustený, môžete zmeniť poradie jazykov aby ste sa uistili, že aplikácie spustené po OmegaT nebudú ovplyvnené zmenou jazyka.

Sady znakov a Písma pre jazyk

OmegaT poskytuje nastavenia písma zobrazenia textu.

Pre povolenie prekladu medzi dvoma jazykmi, ktoré používajú rôzne sady znakov, vyberte písmo, ktoré obsahuje znakové sady obidvoch abecied. V istých prípadoch môže byť potrebná inštalácia dodatočných písiem.

Iné písmo zobrazovania môže byť vybraté cez dialóg Písmo zobrazenia. Otvorte ho cez položku v menu Nastavenia > Písmo zobrazenia.... Typ písma a veľkosť môžu byť zmenené z dialógu.

Všimnite si, že vybraté písmo sa vzťahuje iba na zobrazenie používateľského rozhrania (tj. okná Editora a Prehliadača zhody a slovníka. Nemá to žiaden vplyv na písma použité v zdrojových a cieľových dokumentoch. Písma dostupné pre výber sú tie, ktoré sú dostupné na vašom systéme.

Varovanie! Môže trvať značný čas kým OmegaT aktualizuje zobrazovanie po zmene nastavenia písma, zvlášť keď je otvorený veľký projekt. Tiež, niektoré písma sa pre niektoré páry jazykov správajú lepšie ako pre iné. Vo všeobecnosti, je lepšie meniť písmo pred otvorením projektu.
