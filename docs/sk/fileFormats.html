Podporované formáty súborov

OmegaT 1.6.1 má zabudovanú podporu nasledujúcich typov súborov:

  • súbory OpenDocument (tiež známe ako dokumenty OpenOffice.org alebo StarOffice)
  • HTML a správne vytvorené XHTML
  • súbory DocBook
  • súbory PO
  • Obyčajné textové súbory
  • lokalizačné súbory Java (zdrojové balíčky)
  • súbory INI (text Kľúč=Hodnota)
  • súbory HTML Help Compiler

S trochou ručnej práce môžete tiež prekladať súbory Microsoft Office. Jednoducho použite OpenOffice.org na prevod vašich súborov Microsoft Office do a z príslušných súborov OpenDocument. Pre súbory Microsoft Word je ďalším spôsobom ich uloženie ako HTML pomocou samotného programu Microsoft Word.
