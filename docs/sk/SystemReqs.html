Systémové požiadavky

Java Runtime Environment

Program OmegaT vyžaduje mať nainštalované prostredie Java Runtime Environment. (tiež známe ako JRE). V podstate, ak môžete na vašom systéme spúšťať programy Java tak budete môcť spustiť aj OmegaT. Prostredia Java Runtime Environment sú dostupné pre nasledujúce platformy: Windows 95/98/2000/ME/XP, UNIX, Linux, a Solaris (x86 aj Sun SPARC). Mac OS X ho má predinštalované.

Toto vydanie OmegaT vyžaduje prostredie JRE verzie 1.4 alebo novšie.

Pokiaľ si je tím projektu OmegaT vedomý, OmegaT NEbeží na nasledujúcich platformách:

  • Windows NT; NT implementácia Javy obsahuje chyby
  • Mac OS 9; jediné prostredie Java Runtime Environment preň dostupné (známe ako MRJ) je príliš staré a neposkytuje požadovanú funkčnosť.
