Támogatott állományformátumok

Az OmegaT 1.6.2 eleve támogatja az alábbi típusú állományok fordítását:

  • Microsoft Open XML állományok
  • OpenDocument állományok (más néven OpenOffice.org vagy StarOffice dokumentumok)
  • HTML és jól formált XHTML
  • DocBook állományok
  • PO-állományok
  • Egyszerű szöveg
  • Java lokalizációs állományok (nyelvi forrásszövegek)
  • INI-nyelvi forrásszövegek (kulcs=érték szövegek)
  • HTML Help Compiler állományok

Némi átalakítással fordíthat Microsoft Office állományokat is (a Microsoft Open XML formátumban mentetteken túl). Az OpenOffice.org segítségével Microsoft Office állományait oda és vissza alakíthatja a megfelelő OpenDocument formátumba / formátumból. Microsoft Word állományok esetén másik megoldás, ha átalakítja a szöveget HTML-be a Microsoft Word használatával.
