Системные требования

Для работы OmegaT требуется Java Runtime Environment (среда исполнения Java). Как правило, если в системе есть Java Runtime Environment (также называемая J2RE), OmegaT может функционировать. Java Runtime Environment реализована на следующих платформах: Windows 95/98/2000/ME/XP, UNIX, Linux, Solaris (для x86 и Sun SPARC), и Macintosh OS X.

По нашим сведениям, OmegaT не работает в Windows NT из-за дефекта в реализации Java для NT.
