Ondersteunde bestandsformaten

OmegaT 1.6.2 ondersteunt van nature het vertalem van bestanden van de volgende typen:

  • Microsoft OpenXML-bestanden
  • OpenDocument-bestanden (ook bekend als OpenOffice.org of StarOffice documenten)
  • HTML en goedgevormde XHTML
  • DocBook-bestanden
  • PO-bestanden
  • Platte-tekst bestanden
  • Java vertaalde bestanden (bronbundels)
  • INI-bestanden (Key=Value tekst)
  • HTML Help Compiler-bestanden

Met een beetje handwerk kunt u ook Microsoft Office-bestanden vertalen (andere dan die welke zijn opgeslagen in het Microsoft Open XML-formaat). Gebruik eenvoudigweg OpenOffice.org om uw Microsoft Office-bestanden te converteren naar en vanuit overeenkomende OpenDocument. Een andere manier voor Microsoft Word-bestanden is om ze als HTML op te slaan met behulp van Microsoft Word zelf.
