Podržani formati datoteka

OmegaT 1.6.1 izvorno podržava prevođenje sljedećih tipova datoteka:

  • Microsoft Open XML datoteke
  • OpenDocument datoteke (takođe poznate kao OpenOffice.org ili StarOffice documents)
  • HTML i dobro formirane XHTML
  • DocBook datoteke
  • PO datoteke
  • Plain text datoteke
  • Datoteke Java localization files (resource bundles)
  • Datoteke INI (Key=Value text)
  • Datoteke Help Compiler

Uz malo truda takođe možeš prevoditi i datoteke Microsoft Office. Jednostavno, za konverziju tvojih datoteka tipa Microsoft Office, radi pretvaranja u odgovarajuće datoteke OpenDocuments, koristi OpenOffice.org, kao i natrag iz njih ka formatima MS Office. Za datoteke Microsoft Word, drugi način predstavlja njihovo pretvaranje u HTML kroz Microsoft Word.
