Para empezar - un proyecto de ejemplo

El procedimiento que se detalla aquí tiene fines didácticos; no es preciso seguirlo estrictamente en la práctica.

  • Ejecute OmegaT.

  • Cree un nuevo proyecto seleccionando Archivo > Crear nuevo proyecto en la barra de menús.

  • Aparecerá un diálogo para desplazarse por los archivos. En este diálogo, seleccione un directorio adecuado en el cual colocar su proyecto.

  • En el campo Nombre de archivo, introduzca un nombre (p.ej. ABC) para el proyecto y confírmelo.

  • Aparecerá un nuevo diálogo que indicará esta vez los directorios del proyecto. Como puede ver en el diálogo, OmegaT le propone crear un nuevo directorio con el nombre que ha seleccionado (p.ej. ABC), y dentro del mismo otros cinco subdirectorios. Puede modificar las ubicaciones de estos directorios predeterminados, pero para los propósitos de este ejemplo, confirme los valores propuestos (los archivos y directorios del proyecto se explican en detalle aquí.)

  • Introduzca los códigos de idioma y país de los idiomas de origen y destino, por ejemplo DE-CH para alemán (Suiza), EN-UK para inglés (Reino Unido). Los códigos en sí no son especialmente importantes a menos que pretenda intercambiar las memorias de traducción con clientes o colegas, en cuyo caso debe seguir la norma requerida (normalmente ISO).

  • OmegaT crea ahora un directorio principal (también denominado "directorio raíz del proyecto") con el nombre que seleccionó previamente (p. ej. ABC). Bajo el directorio raíz del proyecto se encuentran cinco subdirectorios: /glossary, /source, /omegat, /target y /tm.

  • Localice un archivo que desee traducir. Guárdelo como archivo de texto con la extensión *.txt, y colóquelo en /source.

  • Abra el proyecto que acaba de crear con Archivo > Abrir. OmegaT busca automáticamente en el último directorio que usted ha abierto. El directorio raíz del proyecto aparece con el icono de OmegaT, lo que indica que se trata de un proyecto.

  • Abra el proyecto seleccionando en primer lugar el directorio de proyecto marcado con el icono (haga doble clic o seleccione y confirme con "Abrir"), y a continuación el archivo omegat.project. No intente abrir ningún otro archivo, como por ejemplo su archivo con el texto de origen: ¡no funcionará!

  • OmegaT abre el proyecto, mostrando el texto del archivo en la ventana principal de OmegaT.

  • Sitúe el cursor en el campo de destino del primer segmento, entre las etiquetas y . Teclee la traducción y borre el texto original. Pulse Intro para confirmar.

  • Repita este proceso con cada segmento hasta que haya traducido todo el texto.

  • Seleccione ahora Archivo > Guardar, y a continuación Archivo > Compilar. Ahora puede cerrar OmegaT (Archivo > Salir).

  • Encontrará el archivo traducido en el directorio /target.
