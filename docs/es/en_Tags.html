Edición de textos con etiquetas (con formato)

Trabajar con etiquetas

OmegaT muestra el formato interno de los archivos soportados (por el momento HTML, XHTML y OpenOffice.org/StarOffice) como etiquetas. Las etiquetas que se muestran en OmegaT no son las mismas utilizadas por las aplicaciones originales; OmegaT las convierte internamente, y las reconvierte al compilar.

Hay que tener cuidado con las etiquetas. Si se modifican accidentalmente, el formato del archivo final puede corromperse. En el peor de los casos, puede que no sea posible abrir el archivo: esto ocurre en particular con los archivos OpenOffice.org.

Aunque no se puede añadir formato en OmegaT, se tiene un cierto nivel de control sobre el formato existente.

Manipulación de etiquetas en archivos OpenOffice.org

Las etiquetas de OpenOffice.org no admiten cambios. No se pueden borrar, duplicar ni cambiar de orden. Sí se pueden:

  • mover las etiquetas a otro lugar del segmento; deben, sin embargo, mantener el mismo orden;

  • anular las etiquetas colocando una junto a otra. Por ejemplo, si cambia

    Este texto está en <f0>cursiva</f0>
    a
    Este texto está en <f0></f0>cursiva

    elimina de hecho la cursiva.

Manipulación de etiquetas en archivos HTML

  • El inicio del formato viene marcado por una etiqueta de apertura de la forma <xx>, y el final del formato por su correspondiente etiqueta de cierre </xx>. Cualquier cambio que se introduzca en las etiquetas de formato debe aplicarse a ambas etiquetas de la pareja.

  • Se pueden borrar las etiquetas. Por ejemplo, para eliminar la cursiva de

Este texto está en <f0>cursiva</f0>

borre simplemente las etiquetas <f0> y </f0>.

  • Se pueden duplicar las etiquetas. Por ejemplo,

Este texto está en <f0>cursiva</f0>

aplica el formato de la palabra "cursiva" a la palabra "éste".

  • Se puede cambiar el orden de las etiquetas. Por ejemplo,

Este texto está en <f0>cursiva</f0>, este texto está <f1>subrayado</f1>

puede cambiarse a

Este texto está en <f0>cursiva</f0>

  • Se pueden anidar las etiquetas. Por ejemplo:

Este texto está en <f0>cursiva</f0>, este texto está <f1>subrayado</f1>

puede cambiarse a

Este texto está en <f0>cursiva y <f1>este texto está también subrayado</f1></f0>

  • No debe solapar las etiquetas. Por ejemplo:

Este texto está en <f0>cursiva, este texto está también <f1>subrayado</f0>, este texto está subrayado pero no en cursiva</f1>

no está permitido y corromperá su archivo si lo intenta utilizar. Si inserta una etiqueta de apertura y abre a continuación una segunda etiqueta de apertura, debe insertar la etiqueta de cierre de la segunda etiqueta de apertura antes de la etiqueta de cierre de la primera.

Validar las etiquetas

La función de validación de etiquetas (Herramientas > Validar etiquetas) detecta las etiquetas que han sufrido cambios (deliberados o accidentales) e indica los segmentos que se han visto afectados.
