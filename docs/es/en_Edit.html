Funciones de edición

Las funciones de edición de OmegaT pertenecen al campo del editor, que es el texto de destino del segmento activo. Las operaciones de edición se hallan restringidas a este campo, es decir, sólo el texto de destino del segmento activo puede ser editado en todo momento. El texto de cualquier campo de texto situado fuera del campo del editor puede ser sin embargo marcado y copiado para ser pegado.

OmegaT soporta las operaciones habituales con el ratón, las teclas de cursor y los atajos de éstas, más los atajos para las operaciones de marcar, cortar y pegar. Estos atajos pueden variar dependiendo del sistema operativo.

Las operaciones Deshacer y Rehacer son posibles con Ctrl-Z y Ctrl-Y, respectivamente.

No se puede añadir ningún formato al texto que se está traduciendo. El formato de un archivo con formato (en la actualidad, StarOffice/OpenOffice.org y HTML son los formatos soportados), se mantiene, y se cuenta con un margen de control limitado sobre él (consulte Etiquetas para más detalles). Para añadir formato, inserte un marcador como "##" y déle formato al texto en otra aplicación como OpenOffice.org tras compilar.
