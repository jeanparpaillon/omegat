Requisitos del sistema

OmegaT requiere el Java Runtime Environment. Básicamente, si su sistema puede ejecutar el Java Runtime Environment (también conocido como J2RE), podrá ejecutar OmegaT. El Java Runtime Environment está disponible para las siguientes plataformas: Windows 95/98/2000/ME/XP, UNIX, Linux, Solaris (x86 y Sun SPARC) y Mac OS X.

Hasta donde sabemos, OmegaT no puede ejecutarse en Windows NT debido a un error en su implementación de Java.
