Instalación y configuración

Instalación

La instalación consta de tres pasos. Uno, instalar el J2RE (si no lo está ya). Dos, instalar OmegaT. Tres, configurar. La instalación del J2RE y de OmegaT puede realizarse en cualquier orden. La configuración no es esencial, pero permite ejecutar la aplicación con mayor facilidad.

Instalación del Java Runtime Environment

El Java Runtime Environment (J2RE) le permite ejecutar en su sistema cualquier programa escrito en Java. Está disponible tanto como producto independiente como parte integrante del kit de desarrollo de Java.

Java está asimismo disponible como complemento (plug-in) para los navegadores de Internet. Esta versión de Java no le permite ejecutar OmegaT.

JavaScript es un producto totalmente diferente y técnicamente no está en modo alguno relacionado con Java.

Si desea utilizar OmegaT, el J2RE debe estar instalado en su sistema. OmegaT no funcionará sin él. No intente que OmegaT funcione sin el J2RE. ¡Estará perdiendo el tiempo!

Si no está seguro de que el J2RE esté instalado en su sistema, pruebe simplemente a instalar OmegaT para ver si funciona (consulte más abajo la instalación de OmegaT). Si no ocurre nada, o si recibe mensajes de error, es casi seguro que el Java Runtime Environment no esta instalado. En este caso, consígalo e instálelo.

Usuarios de Windows: pueden comprobar qué versión del Java Runtime Environment está instalada (en caso de estarlo) con Inicio > Configuración > Panel de control. Si el J2RE está ya instalado, verán un panel de control denominado "Java(TM)2 Runtime Environment, Standard Edition 1.4.2_04" o similar.

Usuarios de Linux: tecleen simplemente java -version en la línea de mandatos. Se obtendrá la versión instalada (suponiendo, naturalmente, que esté instalada en el sistema).

Se puede descargar el J2RE desde los enlaces que se encuentran más abajo. Es gratuito (observe las condiciones de la licencia), pero las convenciones de denominación pueden hacer que el encontrar la versión adecuada resulte un tanto confuso. Necesitará la versión indicada para su plataforma (Linux, Windows, etc.). El tamaño del J2RE es de unos 14 MB.

(Nota: a fin de confundir a sus clientes, Sun cambia con regularidad el nombre de sus productos. Así, todas las versiones del JRE a partir de la 1 se conocen como "Java 2", de ahí que el Java Runtime Environment sea ahora el J2RE. El Java Runtime Environment puede ser asimismo conocido como Java Virtual Machine, o JVM, y en Macintosh, puede conocerse como MRJ. El kit de desarrollo se ha denominado Java Developer's Kit (JDK), Software Developer's Kit (SDK), y ahora también J2SE, J2EE y J2ME, según la versión. Al menos, eso es lo que creo, si bien me incluyo entre los confundidos.)

OmegaT 1.4 debería funcionar con cualquier versión del J2RE a partir de la 1.2. Sin embargo, recomendamos obtener la última versión del J2RE, ya que las versiones más antiguas de Java pueden no soportar ciertas características.

Como alternativa al J2RE, puede instalarse el kit de desarrollo. El kit de desarrollo contiene su propia copia del J2RE, y también permite modificar OmegaT.

Los usuarios de Windows y Linux pueden descargar el Java Runtime Environment de:

Sun (todas las plataformas)

Blackdown (Linux)

Los usuarios de Macintosh OS X deberían encontrar el Java Runtime Environment ya instalado.

Se sabe que las versiones anteriores de OmegaT pueden ejecutarse en el MRJ para Macintosh OS 9. La versión 1.4 puede que funcione pero no se ha probado en esta plataforma.

Instale el J2RE siguiendo las instrucciones que lo acompañan.

Instalación de OmegaT

Extraiga simplemente los contenidos del archivo en el directorio de su elección. El archivo puede estar en formato *.zip o *.tar.gzip. Los usuarios de Linux y de Mac (Stuffit) no deberían tener problemas con ninguno de los dos; los usuarios de Windows pueden no tener una utilidad para el último.

Ejecutar la aplicación

Windows y Mac OS X

Haga clic en el archivo OmegaT.jar en el administrador de archivos.

Linux

Ejecute java -jar OmegaT.jar desde la línea de mandatos. Observe la tipografía de OmegaT: las mayúsculas son importantes.

Si no ha configurado Java para detectar la ubicación de los archivos de OmegaT, debe en primer lugar desplazarse al directorio que contiene el archivo OmegaT.jar.

Si no fijado la variable de la ruta de Java, debe especificar la ruta completa. Si no ha hecho nada de lo anterior, su rutina de ejecución podría parecerse a la siguiente:

cd /home/usuario/OmegaT
/home/usuario/j2re1.4.2/bin/java -jar OmegaT.jar

Configuración (sólo Linux)

Si está utilizando Linux pero no se encuentra aún muy familiarizado con este sistema operativo, puede hacer que la ejecución de OmegaT sea más fácil de la siguiente forma:

Cree un guión de shell en un archivo de texto con el siguiente contenido:

#!/bin/bash
cd <OmegaT>
<java>/java -jar OmegaT.jar

en donde <OmegaT> es el directorio que contiene el archivo OmegaT.jar y <java> es el directorio que contiene el binario del J2RE (el archivo denominado java). Si ha fijado la variable de ruta de java, no necesita especificar este directorio, y java -jar OmegaT.jar es suficiente. Es necesario especificar el directorio si no se ha fijado la variable de ruta de Java, y también si su sistema tiene más de una versión de Java instalada y debe asegurarse de que OmegaT sea ejecutado por la versión adecuada.

Guarde el archivo de texto en su escritorio como OmegaT, y cambie sus permisos para hacerlo ejecutable (en GNOME y KDE, por ejemplo, menú contextual > Propiedades > Permisos > Ejecutar). Ahora puede ejecutar OmegaT simplemente haciendo clic sobre el archivo OmegaT. Si quiere añadirle un icono del programa, lo encontrará en /OmegaT/images.
