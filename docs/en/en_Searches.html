Keyword and exact searches

OmegaT provides a powerful range of search functions.

Call up the Search function with Ctrl+F and enter the word or phrase you wish to search for in the “Search for” box.

Alternatively, mark a word or phrase in the editing field (target text of the active segment) and hit Ctrl+F. The word or phrase is entered in the “Search for” box automatically in this case.

Select the type of search using the radio buttons (keyword and exact searches) and check boxes (search TMs/search files).

All types of search are non-case-sensitive, i.e. a search for “abc” finds instances of abc, ABC, Abc, etc.

Keyword searches

Select “Keyword search” and enter any number of individual words, in any order, and OmegaT displays a list of all segments in which all the words occur. Keyword searches are similar to a search “with all of the words” in an Internet search engine such as Google (AND logic).

Keyword searches can only find whole words, not parts of words or incomplete words.

Exact searches

An exact search looks for a phrase, i.e. if several words are entered, they are found only if they occur in exactly the same sequence. Exact searches are similar to an “exact phrase” search in an Internet search engine such as Google.

Exact searches can search for the search term:

  • in source and target segments in the current project;
  • in source and target segments in the current project and in translation memories (files in /tm);
  • in any files in a format which OmegaT can read (.txt, .html, .xhtml or .sxw), in any selected directory and (if desired) subdirectories.

The wildcard search characters '*' and '?' can be used in exact searches.

  • The character '?' matches any single character,i.e. 'b?d' finds instances of 'bad' and 'bid'.
  • The character '*' matches any character or number of characters, i.e. 'b*d' finds instances of 'bad', 'bid', 'bold' and 'band'.
