Text Navigation

Move to the:

  • next segment by pressing <Enter>, <Ctrl+N>, or <Tab>
    (Tab Advance must be enabled in the Settings menu for use);

  • previous segment by pressing <Ctrl+P>;

  • next untranslated segment by pressing <Ctrl+U>.

Movement to a different segment within the same file is accomplished by dragging the vertical scroll bar of the window up or down, then mouse click inside the desired segment to activate it. Double clicking may be necessary.

To access a different file within the same project, select it in the Project Files window by clicking on the filename of the desired file to open in the editor. By default, the first segment at the beginning of the file will be the active one. Open the project files list window with File > Project Files..., if it is not already visible.

To return to a segment later, insert a marker in the segment. Then it will be possible to find it using the search functionality. It is a good idea to include the segment number with the marker. The reason being that if the segment is a 100% exact match to other segments the marker will be reproduced in all similar segments due to automatic insertion. The only way to locate the right segment is to compare the marker, which now includes a number, to the active segment number (see Searches: Keyword and Exact).

See also: General Hints and Tips.
