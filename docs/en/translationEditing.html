Translation Editing

You translate your files by moving from segment to segment, and editing a single segment at a time. When moving between segments, editing field may be populated with some text or not. For example, editing field is always populated with the existing translation when you move to translated segment, so that you may edit your previous translation.

When you move to a segment that is not yet translated, however, OmegaT may populate the segment with the text you setup it to, so that you edit the way you like it:

  • OmegaT may automatically populate the editing field with the source text. This is extremely useful for the text containing many trade marks and other proper nouns you need to leave intact. Go to Options -> Editing Behaviour... and make sure The source text radio button is selected.
  • Or OmegaT may leave the editing field blank. This option allows you to enter the translation without the need to remove the source text, thus saving you a pair of keystrokes (Ctrl+A and Del). Go to Options -> Editing Behaviour... and select the Leave the segment empty radio button.
  • OmegaT may also insert the translation of the string most similar to the one you are going to translate. Go to Options -> Editing Behaviour..., tick the flag Insert the best fuzzy match, adjust minimal similarity the matches should have to be inserted automaticaly.

In order to setup OmegaT for any of the following, Go to Options -> Editing Behaviour..., and...

  • Make sure The source text radio button is selected. This is the default setting, so most probably it will be there.
  • Select Leave the segment empty radio button.
  • Tick the flag Insert the best fuzzy match and adjust minimal similarity the matches should have to be inserted automaticaly. You may also wish to edit the prefix to distinguish inserted fuzzy matches.

... and click OK.

Text To Store

Documents you need to translate might contain trade marks, names or other proper nouns that have to be the same in translated documents. There are two strategies for segments that contain only such invariate text.

First, you may choose not to translate such segments, though if you may fail to remember these are invariant and translate them later during the translation review, as OmegaT would report these segments as not translated. This is the default.

Second, you may tell OmegaT to store the translation even if it's equal to the source text, so that OmegaT remembers you reviewed the text and decided to translate it the same. Go to Options -> Editing Behaviour... and tick the flag Allow translation to be equal to source.
