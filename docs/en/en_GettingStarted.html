Getting started - an example project

The procedure here is for the purposes of explanation; it need not be followed rigidly in practice.

  • Launch OmegaT.

  • Create a new project by selecting File > Create New Project from the menu bar.

  • A file navigation dialog is displayed. In this dialog, select a suitable directory into which to place your project.

  • In the File name field, enter a name (e.g. ABC) for your project, and confirm.

  • Another dialog is called up, this time indicating the directories for the project. As you can see from the dialog, OmegaT is proposing to create a new directory with the name you selected (e.g. ABC), and below that a further five subdirectories. You can modify the locations of these default directories, but for the sake of the example, confirm the defaults. (The project files and directories are explained in detail here.)

  • Enter the language and locale codes for your source and target languages, e.g. DE-CH for German (Switzerland), EN-UK for English (United Kingdom). The actual codes are not particularly important unless you intend to exchange translation memories with customers or colleagues, in which case you should be careful to follow the standard required (usually ISO).

  • OmegaT now creates a main directory (also referred to as the "project root directory") with the name you selected previously (e.g. ABC). Below the project root directory are five subdirectories: /glossary, /source, /omegat, /target and /tm.

  • Find a file to translate. Save it as a text file with the extension *.txt, and place it in /source.

  • Open the project you have just created with File > Open. OmegaT automatically looks in the directory you last opened. Your project root directory is displayed with the OmegaT icon, indicating that it is a project.

  • Open the project by selecting first the project directory marked with the icon (double-click or select and confirm with "Open"), then the file omegat.project. Do not try to open any other file, such as your source text file - it won't work!

  • OmegaT opens the project, displaying the text of your file in the main OmegaT window.

  • Place the cursor in the target field of the first segment, between the tags <segment 0001> and <end segment>. Type in your translation and delete the original text. Hit Enter to confirm.

  • Repeat this process with each segment until you have translated the complete text.

  • Now select File > Save, then File > Compile. You can then close OmegaT (File > Quit).

  • You will find your translated file in the directory /target.
