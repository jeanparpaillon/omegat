Files to translate

With OmegaT you can directly translate a number of file formats. There are basically two types of file format. The plain text formats and the formatted text formats.

Plain text files

Plain text files contain text only, so their translation is as simple as typing the translation.

Currently, OmegaT supports the following plain text formats:

  • ASCII text (.txt, etc.)
  • Encoded text
  • Java resource bundles (.properties)
  • PO files (.po)
  • INI (key=value) files (.ini)

Such files are not considered as containing more format than "white space" for indentation/alignement purposes.

They are usually modified in text editors and it is generally not possible to have such format retain font/color/margin etc information.

Formatted text files

Formatted text files contain text as well as information such as font type, size, color etc. They are commonly created in word processors or home page editors.

Currently, OmegaT support the following formatted text formats:

  • OpenOffice.org files
  • OpenDocument files
  • (X)HTML files
  • HTML Help Compiler (HCC) files
  • DocBook files

Such file formats are conceived so that they retain formatting information. Such formatting information can be as simple as this is bold or as complex as table data with different font size, color, position etc.

In most translation jobs it is considered important to have the translated document look similar to the original. OmegaT allows you to do this by marking the characters/words that have a special formatting with easy to manipulate tags.

Simplifying the original text formating greatly contributes to reducing the number of tags. Unifying used fonts, font sizes, colors etc should be considered if possible, to simplify the translation and reduce the number of possible tag errors.

File formats specifics

Most of the above file formats are subject to specific handling.

Refer to Supported File Formats and File Filters for specific information related to the handling of all the above file formats.

Other file formats

Other plain text or formatted text file formats may be accessible in OmegaT

Roughly speaking, a number of plain text formats (including LaTex etc) can be translated in OmegaT through conversion to the PO format.

Similarly, a number of formatted text formats (including Microsoft Office files) can be translated in OmegaT through conversion to the Open Document format.

For more information, refer to Supported File Formats.
