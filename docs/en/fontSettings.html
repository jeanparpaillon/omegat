Font settings

This dialog allows you to modify the font OmegaT uses in the following windows:

  • OmegaT main window (Editor, Match viewer, GLossary viewer)
  • Search window
  • Tag validation window

It is called via the Options â†’ Font... item in the Main menu.

The dialog contains:

  • Drop-down to select one of the fonts available on your machine
  • Spin edit to change font size
  • An area with the same text to preview the selected font immediately

Warning! It may take a noticeable amount of time for OmegaT to update the display after you have changed the font setting, especially when a big file with many segments is open in the editor. Also, some fonts behave better for some language pairs than for others.
