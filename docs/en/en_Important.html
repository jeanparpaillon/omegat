Important

OmegaT is a robust application. It is not prone to crashing or to loss or corruption of data. However, you should take precautions against data loss when using OmegaT, just as with any other application.

  • Use OmegaT on short texts until you are familiar with it.
  • Read Saving, revising and compiling your translation carefully.
  • Most users will be able to get started with the aid of these Help texts. Should you run into problems, post a message in the OmegaT user forum located at http://groups.yahoo.com/group/omegat.

Remember also that OmegaT creates a backup of project's translation memory to project_save.tmx.bak file in the project's /omegat subdirectory every time when a project is saved or compiled.
