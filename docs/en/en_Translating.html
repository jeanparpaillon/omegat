The translation process

Open a project and translate by overwriting the text in the active segment.

Hit Enter or Ctrl+N to confirm your translation and proceed to the next segment. If you need to keep Enter for other use, for example, to engage it in a tool for input of complex characters, you can alternatively use Tab. Using Tab is an option that should be activated by clicking on the Preferences > Use TAB to advance menu item.

Upon opening a project, the first segment of it automatically becomes active. To quickly jump to the next untranslated segment of the project, hit Ctrl+U or choose Edit > Next untranslated menu item.

After translating all the segments, compile the project by choosing the File > Compile translation menu item.

After compilation, you can review the results checking the files in the /target subdirectory of your project (for details on files and directories see Project files and directories). If you are not satisfied with the results, just renew translation according to the described procedure (see also Saving, revising and compiling your translation, General hints and tips and Keyboard shortcuts).
