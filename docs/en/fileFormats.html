Supported File Formats

OmegaT 1.6.0 natively supports translating the following types of files:

  • OpenDocument files (also known as OpenOffice.org or StarOffice documents)
  • HTML and well-formed XHTML
  • DocBook files
  • PO files
  • Plain text files
  • Java localization files (resource bundles)
  • INI files (Key=Value text)

With a bit of hand-work you may also translate Microsoft Office files. Simply use OpenOffice.org to convert your Microsoft Office files to and from corresponding OpenDocuments. For Microsoft Word files another way is to save them to HTML using Microsoft Word itself.
