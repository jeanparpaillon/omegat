Supported File Formats

OmegaT 1.7.3 natively supports and recognizes the following file types:

    • Microsoft Open XML files
    • OpenDocument files (also known as OpenOffice.org or StarOffice documents)
    • HTML and well-formed XHTML
    • DocBook files
    • PO files
    • Plain text files
    • Java localization files (resource bundles)
    • INI files (Key=Value text)
    • HTML Help Compiler files
    • Okapi monolingual XLIFF files

With a bit of hand-work you may also translate Microsoft Office files (stored for instance as DOC or RTF file). Simply use OpenOffice.org to convert your Microsoft Office files to and from corresponding OpenDocuments. Note also that Microsoft Word files can be transformed to HTML using Microsoft Word itself as well.
