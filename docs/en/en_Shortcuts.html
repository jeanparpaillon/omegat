Keyboard shortcuts

Ctr+O Open existing project
Ctr+S Save translation memory
Ctr+Q Exit OmegaT
Enter, Ctrl+N, Tab* Jump to next segment
Ctr+P Jump to previous segment
Ctr+F Search translation memory
Ctr+R Overwrite translation (replace text in active segment with selected fuzzy match)
Ctr+I Insert translation (insert selected fuzzy match into active segment at cursor position)
Ctr+1, 2, etc. Select fuzzy match #1, fuzzy match #2, etc.
Ctr+L Display list of project source files
Ctr+U Jump to next untranslated segment

*Using Tab is an option that should be activated by clicking on the Preferences > Use TAB to advance menu item.
