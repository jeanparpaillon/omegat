What's new in 1.4.4

  • Easy Help system localization (and Belarusian, French, Russian and Spanish l10ns)
  • Separate source distribution of OmegaT
  • Ability to jump to next not yet translated segment (for details look at The translation process)
  • New Fuzzy Matching Engine:
    • enhanced matching for Japanese
    • increased number of fuzzy matches
  • Refactored HTML and XML file handlers to improve speed and stability
  • Some refactorings of file handlers
  • Support for glossaries in UTF-8 encoding (for details look at Translation memory and glossary file formats)
  • Better HTML handling, including better encoding recognition and meta-data transfer to target files
  • No more multiple help file viewers
  • Empty segments are not displayed
  • Reliable font change
