What to translate

The localisation package consists of three parts, namely:

  • The user interface file:
    src\org\omegat\Bundle.properties
    A total of about 350 "sentences" (of which about half are one or two words long)
  • The Instant Start:
    docs\en\InstantStart_NoNav.html
    A total of about 100 sentences.
  • The user manual:
    docs\en\[a whole bunch of HTML files]
    A total of about 2000 sentences.

A new language is included in the official OmegaT release once the user interface and the Instant Start have been translated, although ideally we would like to have all three parts translated.

How to translate and test the translation

All three parts of the localisation package can be translated using OmegaT.

The user interface file

src\org\omegat\Bundle.properties

This is a Java properties file. It contains the text for the menus and dialog boxes within OmegaT.

Testing it requires updating the OmegaT.jar file, which can be a bit tricky. If you struggle, you're welcome to ask the others on the localisation mailing list if anyone might be willing to help you.

The following instructions are for Microsoft Windows:

  1. Rename the Bundle.properties file to Bundle_xx.properties (xx being your language code).
  2. Do a search of your hard disk for the file "jar.exe".
  3. In the folder which contains jar.exe, create a new, temporary folder, and copy the OmegaT.jar file from your OmegaT installation into it.
  4. Open a command prompt (also known as a terminal window or a DOS window), and navigate to that temporary folder.
  5. Type:
    ..\jar.exe xvf OmegaT.jar
    This extracts the JAR file into two subfolders "org" and "META-INF".
  6. Delete the OmegaT.jar file.
  7. Add your Bundle_xx.properties file in the subfolder org/omegat/, where the original Bundle.properties is also present.
  8. In the command prompt, type:
    ..\jar.exe cvfm OmegaT.jar META-INF/MANIFEST.MF org
    This recreates the JAR file.
  9. Copy the newly created JAR file back to its original installed position.

In short, "jar xvf OmegaT.jar" extracts it, and "jar cvfm OmegaT.jar META-INF/MANIFEST.MF org" puts it back together.

Then run OmegaT using the following command:

java.exe -Duser.language=xx -jar OmegaT.jar

The Instant Start

docs\en\InstantStart_NoNav.html

The Instant Start is an ordinary HTML file. It is the first thing a user sees when he runs OmegaT, and contains a very, very short tutorial for getting started with OmegaT.

Testing it is easy. Simply open the file in any web browser. Alternatively, you can replace the original InstantStart_NoNav.html file in your existing OmegaT's installation with the translated one, and then run OmegaT.

The User Manual

docs\en\[a whole bunch of HTML files]

The user manual consists of a number of ordinary HTML files. These files are viewed from within OmegaT with a simple helpfile viewer, but they can also be viewed in any web browser.

Testing it is easy. Simply open the files in any web browser. Alternatively, you can replace the original user manual files in your existing OmegaT's installation with the translated ones, run OmegaT, and press F1.
