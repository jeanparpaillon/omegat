Note! Throughout the manual, references to the free and open source office suite OpenOffice.org are made. Depending on the context, the references point to different things.

  • In a file format context, the reference points to the file format used by OpenOffice.org before version 2.0 (currently version 1.1.5). The OpenOffice.org standard file format as of version 2.0 is the OASIS standard OpenDocument. Both formats are extremely similar and OmegaT supports them equally.
  • In an application context, the reference points to applications which code base is borrowed from OpenOffice.org and which thus offer extremely similar functions. While OpenOffice.org is available for a wide number of operating systems (including Windows, Linux and MacOSX) "sister" applications have sometimes a narrower operationg system support. The two most well known sister applications are StarOffice (available on Windows and Linux) and NeoOffice (available on MacOSX).

Main news in recent versions

  • Sentence Segmenting, providing better translation reuse via a fine-grained segmentation.
  • Massive rewrite of HTML and XHTML file filters, providing you a rock-solid HTML/XHTML translation experience.
  • User configurable file filters for advanced control of source to target (translated) file encodings and naming conversion.
  • Split pane in Match and Glossary Viewer that allows for adjustable resizing via dragging or clicking with the mouse.
  • Menubar and menu changes. Some renaming and rearrangement for better use. Help menu includes access to a new About dialog as well as the Users Manual.
  • Improved user interface text strings throughout.
  • Log file (log.txt) for tracking operations and errors.
