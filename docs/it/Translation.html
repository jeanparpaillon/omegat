Translation Process

To translate text just overwrite it in the active segment's editing field.

Press <Enter> or <Ctrl+N> to accept a translation and proceed to the next segment. If <Enter> is needed for another purpose, like using it in a tool for the input of complex characters, it is possible to use <Tab> as an alternative. To activate this functionality, enable the TAB Advance menu item checkbox in the Settings menu.

When a project is initially opened the the active segment defaults to the first segment of the first file in the Project Files list. To jump quickly to the next untranslated segment of the project, press <Ctrl+U> or choose the Next Untranslated Segment menu item from the Edit menu.

In some case the source and target segments will be the same even after translation. In such cases OmegaT will consider these segments as untranslated since there is no way to know if the segments' translations have been completed. Segments are considered to be untranslated when source and target segments are the same.

Create translated files at anytime during the translation process by compiling the project documents with the Create Target Documents menu item from the File menu. It is not necessary to have all files and segments translated before using this function. The translated segments will be written to the files and any remaining untranslated segments will be written out untouched. This can be useful, for example, to perform a partial review of the translation. The target files are created in the /target subdirectory of the project (for details on files and directories see Project Files and Directories).

Please be aware that modifying target files in an outside application without updating the segments from within OmegaT is dangerous because the next compilation performed in OmegaT will only take into consideration modifications made internally. Any changes made externally will be overwritten and lost in this case. If the translation of the target files is not satisfactory, redo the translation according to the previously prescribed procedure (see also Translation: Save, Revision, and Compilation, General Hints and Tips and Keyboard Shortcuts).
