Onartutako fitxategi-formatuak

OmegaT 1.6.2 bertsioak ondoko fitxategi-motak onartzen ditu itzulgai gisa:

  • Microsoft-en Open XML fitxategiak
  • OpenDocument fitxategiak (OpenOffice.org edo StarOffice dokumentu modura ere ezagutzen dira)
  • HTML eta ondo egituratutako XHTML
  • DocBook fitxategiak
  • PO fitxategiak
  • Testu hutseko fitxategiak
  • Javaren lokalizazio fitxategiak (baliabide-paketeak)
  • INI fitxategiak ('gako=balio' formatua)
  • HTML laguntza konpilatzeko fitxategiak

Eskuz pixka bat aldatuz, Microsoft Office-n fitxategiak ere (alegia, Microsoft-en Open XML formatuan gorde ez direnak) itzuli ditzakezu. Gauza bakarra egin behar duzu: OpenOffice.org erabili zure Office fitxategiak OpenDocument formatura bihurtzeko. Microsoft Word-en fitxategietarako badago beste bide bat: gorde fitxategia HTML formatuan Microsoft Word bera erabiliz.
