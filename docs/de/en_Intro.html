Was ist eine Segmentenpaardatenbank?

Die Segmentenpaardatenbank wird während der Übersetzung von Dokumenten von einer Sprache in eine andere verwendet. Sie dient dazu, Textteile in einer Datenbank (der ¨Segmentenpaardatenbank¨) zu speichern, so dass man sofort sehen kann, wie ein Text in Vergangenheit übersetzt wurde, wenn man dasselbe Textstück (ein ¨Segment¨) oder einen Teil davon nochmals vorfindet. Damit eine Segmentenpaardatenbank wirklich nützlich ist, muss sie lange genug verwendet werden, um eine Datenbank mit Segmenten und deren dazugehörige Übersetzungen aufzubauen.

Was ist besonders bei OmegaT?

Es gibt viele CAT-Tools auf dem Markt. Die besonderen Eigenschaften von OmegaT sind:

  • es funktioniert mit Hilfe des Java Runtime Environments und kann somit auf verschiedenen Betriebssystemen laufen, dies inklusive Windows, Linux, und Mac OS X;

  • es ist kostenlos;

  • es ist Open Source, was bedeutet, dass Anwender das Programm frei weitergeben können und es ändern können;

  • es wurde mit dem Hauptgedanken der Geschwindigkeit entwickelt, anstatt komplexe Datenformatierung und vielen Dateiformaten;

  • es unterstützt das OpenOffice.org Dateiformat direkt.
