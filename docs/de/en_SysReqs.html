Systemanforderungen

OmegaT benötigt das Java Runtime Environment. Wenn das Java Runtime Environment (auch als J2RE bekannt) auf Ihrem System läuft, dann kann es in der Regel auch mit OmegaT arbeiten. Das Java Runtime Environment gibt es für folgende Betriebssysteme: Windows 95/98/2000/ME/NT/XP, UNIX, Linux, Solaris (sowohl x86 als auch Sun SPARC) und Macintosh OS X.

Soweit wir wissen, arbeitet OmegaT nicht auf Windows NT aufgrund eines Bugs in der NT-Java-Implementierung.
