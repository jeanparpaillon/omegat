Podporované formáty souborů

OmegaT 1.6.2 má zabudovanou podporu následujících typů souborů:

  • soubory Microsoft Open XML
  • soubory OpenDocument (také známé jako dokumenty OpenOffice.org nebo StarOffice)
  • HTML a správně vytvořené XHTML
  • soubory DocBook
  • soubory PO
  • Soubory v prostém textu
  • lokalizační soubory Java (zdrojové balíčky)
  • soubory INI (text Klíč=Hodnota)
  • soubory HTML Help Compiler

S trochou ruční práce můžete také překládat soubory Microsoft Office (jiné než takové, které jsou uložené ve formátu Microsoft Open XML). Jednoduše použijte OpenOffice.org na převod Vašich souborů Microsoft Office do a z příslušných souborů OpenDocument. Pro soubory Microsoft Word je dalším způsobem jejich uložení jako HTML pomocí samotného programu Microsoft Word.
