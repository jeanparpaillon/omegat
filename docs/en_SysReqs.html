System requirements

OmegaT requires the Java Runtime Environment. Essentially, if your system will run the Java Runtime Environment (also known as the J2RE), it will run OmegaT. The Java Runtime Environment is available for the following platforms: Windows 95/98/2000/ME/XP, UNIX, Linux, Solaris (both x86 and Sun SPARC), and Macintosh OS X.

As far as we are aware, OmegaT does not run on Windows NT owing to a bug in the NT Java implementation.
