Formats de fitxer compatibles

De manera nativa, l'OmegaT 1.6.2 permet la traducció dels següents tipus de fitxer:

  • Fitxers Microsoft Open XML
  • Fitxers OpenDocument (anomenats també documents OpenOffice.org o StarOffice)
  • HTML i XHTML amb un format correcte
  • Fitxers DocBook
  • Fitxers PO
  • Fitxers de text net
  • Fitxers de localització Java (paquets de recursos)
  • Fitxers INI (text del tipus Clau=Valor)
  • Fitxers de compilador d'ajuda HTML

Amb una mica de feina manual, també es poden traduir fitxers del Microsoft Office (que no s'hagin desat en format Microsoft Open XML). Simplement cal que utilitzeu l'OpenOffice.org per a convertir els fitxers del Microsoft Office en fitxers OpenDocument i, després de la traducció, invertir-ne la conversió. Una altra manera de traduir fitxers del Microsoft Word és desant-los en format HTML mitjançant el propi Microsoft Word.
