Translation file formats

OmegaT supports the following formats of files for translation:

  • plain text;

  • html;

  • xhtml;

  • OpenOffice.org 1.0/StarOffice 6.0 or higher.

OmegaT detects the file format automatically based upon the file's extension, so make sure that your files have the extensions .txt, .html and .sxw for plain text, html and OpenOffice.org/StarOffice respectively.

OpenOffice.org has excellent import/export filters for conversion of other file formats, including MS Word, Excel and Powerpoint. Files in these formats can therefore be processed by conversion in OpenOffice.org to the latter's native format, translation in OmegaT, and conversion back again following translation.
